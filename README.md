## ✍️ Example

Variable Example :-

```bash
let name = "John";
console.log("Hello, " + name);
```

Conditional Example :-
```bash
let age = 20;
if (age >= 18) {
  console.log("You are eligible to vote.");
} else {
  console.log("You are not eligible to vote.");
}
```
